# Конвертер HTML файлов анализа в текстовый формат (Node.js)

Этот скрипт извлекает данные из HTML файлов анализа и сохраняет их в текстовом формате.

## Требования

Для работы скрипта необходимы:
- Node.js (версия 12 или выше)
- npm (менеджер пакетов Node.js)

## Установка

Установите необходимые зависимости:

```bash
npm install
```

## Использование

1. Поместите все HTML файлы в папку `result`
2. Запустите скрипт:
```bash
npm start
```
или
```bash
node extract_data.js
```
3. Результаты будут сохранены в папке `extracted_text`

## Структура результата

В текстовых файлах будет следующая информация:
- Метаданные (модель, файл, промпт)
- Дата анализа
- Ответ модели
